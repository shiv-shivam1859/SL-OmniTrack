Detailed Overview of this Project -

Name - SL-OmniTrack
- "SL" stands for my future startup name "Silicon Labs" and OmniTrack is the kits name. This is a kit which includes 2 main things -

1. The ground Station - It comes in a 3d printed enclosure which has a microcontroller , LoRa transceiver , power package and a usb cable going from microcontroller to outside of enclosure for user to connect from PC/Laptop.

2. Main Avoinics bay - It comes in cylindecal shape outline enclosure with diff. diameter option. It has microcontroller , IMU , Barometric sensor , Camera , Micro SD modul (To save flight data) , Power package.

- Both enclosure has switch to power the system , both microcontroller's code make sure's that as soon as microcontroller receive power from battery both of them connects to each other wirelessly.
